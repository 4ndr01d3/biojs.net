BioJS is a JavaScript library of graphical components to represent biological information. A library easy to maintain and develop by identifying minimal functional units that could be used as building blocks for more complex applications. A browser independent JavaScript library easy to integrate in web applications regardless the programming language used to develop the server side.

Das Protein Feature Viewer
This component uses a DASProtein web service that builds the JSON data object used by FeatureViewer component.
Hpa Summary Features
Component to represent Human Protein Atlas summary protein expression information from a DAS XML.
Interactions Bundle D3
This component uses the D3 library and specifically its implementation of the bundle algorithm to represent a network of protein interactions.
Interactions D3
This component uses the D3 library and specifically its implementation of the force algorithm to represent a network of protein interactions.
KeggViewer
Component that renders a KEGGML file and displays a KEGG pathway. It highlights pathway genes according to defined expression values for each experimental condition and allows users to change the threshold parameters for down and up-regulation.
Protein Portafolio
ProteinPortafolio component shows the description of a protein as well as its PDB alignments if there is any. Component shows the PDBLogos of the selected alignment and ables to play with the sequence and the 3D draw of the alignment.
ChEMBLCompound
A component to render ChEMBL compounds.
Rheaction
BioJS component to display Rhea reactions.
Expression Atlas Baseline Summary
This is the ExpressionAtlas component for displaying baseline expression of genes based on RNA-seq experiments in the ExpressionAtlas database.
